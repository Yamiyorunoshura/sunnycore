### 🔄 升級指南

#### 從v1.0.0升級到v2.0.0

1. **備份現有配置**
   ```bash
   cp -r ~/.claude ~/.claude.backup
   ```

2. **執行新安裝腳本**
   ```bash
   chmod +x install.command
   ./install.command
   ```

3. **選擇適當的配置選項**
   - 選擇偏好的語言
   - 配置所需的MCP工具
   - 確認安裝路徑

4. **驗證安裝**
   - 檢查所有檔案是否正確安裝
   - 驗證CLAUDE.md配置是否正確
   - 測試基本工作流程
